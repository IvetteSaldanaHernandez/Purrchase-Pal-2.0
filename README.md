🐾 Purrchase Pal

Purrchase Pal is a social money management platform where friends compete to save money by collectively judging each other’s purchases.
Instead of budgeting alone, users rely on peer accountability + AI insights to make smarter financial decisions.
🧠 The Idea
Most budgeting apps fail because they rely on self-discipline alone.
Purrchase Pal turns saving money into a social game:
Users post purchases they’re considering.


Friends react and judge whether it's a smart buy or a bad financial decision.


AI provides insights about spending patterns.


Users compete to maintain the best spending score.


The result: saving money becomes collaborative, competitive, and fun.
✨ Features
🐱 Cat-themed social feed
 Post purchases and get reactions from friends.


👥 Peer accountability
 Friends vote if a purchase is a Smart Buy or a Bad Buy.


🤖 AI insights
 AI analyzes spending habits and suggests better financial decisions.


🏆 Savings competition
 Leaderboards reward users who consistently make smarter purchases.


📊 Spending score
 Your score reflects how well you resist questionable spending.


🚀 Getting Started
1. Clone the repo
git clone https://github.com/your-repo/Purrchase-Pal-2.0.git

2. Get a Gemini API Key
Go to Google AI Studio
Sign in with your Google account
Click Get API Key
Create a new key
Copy the key
Add the API Key to Your Environment
Create a .env file in the backend folder:
GEMINI_API_KEY=your_api_key_here
3. Install dependencies
cd Purrchase-Pal-2.0/client
cd client
npm install
4. Run the development server
npm run dev
The app will start locally and you can begin using Purrchase Pal.
🛠 Tech Stack
Frontend: React, Vite


Backend: Python, FastAPI, Supabase (Authentication)


AI: Google Gemini API


Database: MongoDB Atlas (PyMongo)


🎮 Example Use Case
You’re about to buy a $90 hoodie.
You post it to Purrchase Pal:
“Do I really need this hoodie?”
Your friends vote:
🐾 Savvy Cat → Smart Buy


🙀 Financial Catastrophe → Bad Buy


AI then tells you:
“You’ve bought 4 hoodies this month. Your friends may be right.”
You reconsider.
Your savings score improves.
Your friends roast you less.
